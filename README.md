# 🏦 نظام إدارة بنك (Bank Management System)

## 🧾 الوصف
هذا مشروع متكامل لمحاكاة نظام بنك باستخدام لغة C++.  
يتيح النظام إدارة العملاء، العمليات المالية، وإدارة المستخدمين مع صلاحيات مختلفة.

---

## 🚀 المميزات

### 👤 إدارة العملاء
- عرض قائمة العملاء
- إضافة عميل جديد
- حذف عميل
- تعديل بيانات عميل
- البحث عن عميل

### 💰 العمليات المالية
- إيداع أموال
- سحب أموال
- عرض إجمالي أرصدة العملاء

### 🔐 نظام المستخدمين
- تسجيل دخول (Login)
- التحقق من اسم المستخدم وكلمة المرور
- دعم الصلاحيات (Permissions)

### 📊 إدارة البيانات
- حفظ البيانات في ملفات نصية:
  - Clients.txt
  - Users.txt
- قراءة وكتابة البيانات باستخدام File Handling

---

## 🛠️ التقنيات المستخدمة
- C++
- File Handling (fstream)
- Vectors
- Structs
- Enums
- String Processing

---

## 📂 ملفات المشروع
Clients.txt → بيانات العملاء
Users.txt → بيانات المستخدمين


---

## 🧠 فكرة العمل
- يتم تسجيل الدخول أولاً باستخدام Username & Password
- يتم تحميل بيانات المستخدمين والعملاء من الملفات
- يتم تنفيذ العمليات من خلال القوائم (Menus)
- يتم تحديث الملفات تلقائيًا بعد أي تعديل

---

## 🎮 القوائم الرئيسية

### 🏠 القائمة الرئيسية
- عرض العملاء
- إضافة عميل
- حذف عميل
- تعديل عميل
- البحث عن عميل
- المعاملات المالية
- إدارة المستخدمين
- تسجيل خروج

### 💳 قائمة المعاملات
- إيداع
- سحب
- عرض إجمالي الرصيد
- العودة للقائمة الرئيسية

---

## ⚙️ طريقة التشغيل
1. تشغيل البرنامج
2. تسجيل الدخول
3. اختيار العملية من القائمة
4. تنفيذ العمليات المالية أو الإدارية
